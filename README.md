# Refined Nyx.cz

> Chrome/Firefox extension for Nyx.cz

## Features

- [x] Markdown support
- [ ] Emoji support
- [x] Notifications in menubar
- [x] Tags
- [x] CMD + Enter submit
- [ ] Load more ajax

## Dev

### How to run

```sh
$ yarn serve
```

This opens Chrome with the extension installed and Webpack running.
Refresh after each change.
